#### 什么是高阶组件

> A higher-order component is a function that takes a component and returns a new component.

上面的解释是React官网给出的，简单来说高阶组件就是一个方法，传入一个组件再返回一个组件。
在这个方法中可以对传入的组件进行包装、添加功能以达到代码复用、抽象逻辑的效果，这也就是高阶组件的主要用途。
